# Детектор накрученных отзывов на Wildberries

![Python](https://img.shields.io/badge/python-3.9+-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

Система автоматического определения фейковых (накрученных) отзывов на маркетплейсе Wildberries с использованием машинного обучения.

## 🎯 Бизнес-задача

Фейковые отзывы искажают репутацию товаров и вводят покупателей в заблуждение. Этот проект демонстрирует подход к автоматической детекции накрученных отзывов на основе текстового анализа и поведенческих паттернов.

## 📊 Данные

- **Источник**: Отзывы с товаров продавца ID 1177893 на Wildberries
- **Объем**: 200+ отзывов с 3+ товаров
- **API**: `https://feedbacks1.wb.ru/feedbacks/v1/{nmId}`

## 🔬 Методология

### Разметка данных (эвристическая)

Поскольку реальной разметки нет, использованы эвристические правила:

**Фейк (1):**
- Рейтинг = 5
- Текст < 150 символов
- 2+ ключевых слова: "отлично", "супер", "рекомендую", "качество", "доставка", "быстро", "огонь", "топ"

**Реальный (0):**
- Рейтинг ≤ 3
- ИЛИ содержит конкретику: цифры, время использования, слова "сломался", "брак", "вернул"
- ИЛИ длинный детальный текст (>300 символов)

**Серые (исключены):** Остальные отзывы

### Признаки (30+)

#### A. Поведенческие (метаданные)
- `hour`, `is_night`, `is_weekend` — временные паттерны
- `likes_ratio`, `has_photo` — взаимодействие
- `text_length`, `word_count` — объем текста

#### B. Лингвистические
- `exclamation_count`, `uppercase_ratio` — эмоциональность
- `unique_words_ratio` — разнообразие лексики
- `fake_keywords_count`, `real_keywords_count` — ключевые слова

#### C. Шаблонность
- `repetitive_bigrams`, `repetitive_trigrams` — повторяющиеся фразы
- `starts_with_positive`, `ends_with_recommend` — шаблонная структура
- `rating_text_mismatch` — несоответствие рейтинга и текста

## 🤖 Модели

### 1. Random Forest
- **Признаки**: 30+ инженерных фич
- **Гиперпараметры**: GridSearchCV (n_estimators, max_depth, min_samples_split)
- **Метрики**: ROC-AUC, Precision, Recall, F1

### 2. BERT (rubert-tiny2)
- **Модель**: `cointegrated/rubert-tiny2`
- **Fine-tuning**: 5 эпох, learning_rate=2e-5
- **Вход**: только текст отзыва

### 3. Ансамбль (Stacking)
- **Базовые**: Random Forest + BERT
- **Мета-модель**: Усреднение вероятностей

## 🚀 Быстрый старт

### Установка

```bash
git clone https://github.com/yourusername/fake_review_detector_wb.git

pip install -r requirements.txt
```

### 1. Сбор данных

```bash

```

## ⚠️ Ограничения и этика

1. **Эвристическая разметка**: Данные размечены автоматически, без ручной проверки. Это может привести к ошибкам в обучающей выборке.

2. **Антибот-защита**: Wildberries ограничивает частоту запросов. В проекте используется задержка 2 сек между запросами.

3. **Образовательные цели**: Парсинг выполнен исключительно для демонстрации навыков Data Science. Для продакшена требуется официальное API.

4. **Обобщаемость**: Модель обучена на данных одного продавца и может не работать на других категориях товаров.

5. **Этика**: Система не должна использоваться для массового удаления отзывов без ручной проверки.
